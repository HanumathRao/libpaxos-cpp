enum enum server::state paxos::detail::quorum::server::state
Enumerator:
state_unknown 

When a server is first created, it's in an unknown state

state_dead 

Used when a connection to the server cannot be established or was lost

state_non_participant 

Used when a connection to a server could be established, but handshake hasn't occured yet.

state_leader 

When a server is known to be the leader of a quorum

state_follower 

When a server is known to be a follower in a quorum

state_client 

Used to identify a client connection. This should never exist within a quorum.
